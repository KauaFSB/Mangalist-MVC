1- Configuração do Git Local:

git config --global user.name

git config --global user.email 

git clone "URL"

2- Criando um Projeto MVC: 

dotnet new mvc -o Pokedex


3- Para executar um projeto: 
dotnet watch run

4- Caso seja necessário restaurar o projeto:
dotnet restore

5- Para limpar dependencias e bibliotecas
dotnet clear
